# Taste of Coffee

세계의 커피와 레시피를 모으는 작은 아카이브 웹사이트.

## Preview

의존성 없는 정적 사이트입니다. 로컬에서 보려면:

```bash
python3 -m http.server 8080
# http://localhost:8080
```

## Structure

```
index.html
assets/
  css/style.css
  js/main.js
```

## Deploy

`index.html`이 루트에 있어 GitHub Pages, Netlify, Vercel 등에 그대로 올릴 수 있습니다.
